# What Is OnionC2?
OnionC2 is a command and control (C2) tool designed for managing remote systems with an emphasis on security, privacy, and secrecy. By utilizing the Tor network, it ensures that communications between the C2 server and remote systems remain secure, anonymous, and available. This makes it well-suited for cybersecurity activities such as red teaming, adversary simulations, and penetration testing.

OnionC2 provides a variety of features, including:
- Execution of shell commands on remote systems.
- Secure file transfers between the server and remote systems.
- Mechanisms for maintaining access on Windows systems.
- End-to-end encryption for all communications.

Additionally, OnionC2 integrates with XENA, offering a user-friendly interface to streamline the management of operations. Keep in mind that XENA's user interface is borrowed, meaning OnionC2 will have its own user interface in the future, custom tailored to OnionC2's capacity.
